# 🌸 She Can Foundation – Full Stack Web Development Project

A modern full-stack NGO website developed as part of the internship selection task for She Can Foundation.

This project showcases frontend development, backend integration, Firebase database management, authentication, responsive UI design, and admin dashboard functionality.

---

# ✨ Project Overview

The website was designed for **She Can Foundation**, a non-profit organization focused on empowering women through:

- Education
- Menstrual Health Awareness
- Digital Skill Development
- Volunteer Programs
- Internship Opportunities

The platform allows users to:

- Send contact messages
- Apply as volunteers
- Apply for internships
- Donate to campaigns
- Contribute professional skills
- Submit partnership enquiries

All submissions are dynamically stored using Firebase Firestore Database.
